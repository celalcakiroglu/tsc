Hyperplane: \(\lbrace \mathbf{x} : \mathbf{w}^T\mathbf{x}=b \rbrace\), \(\mathbf{w,x}\in \mathbb{R}^n\), \(b\in \mathbb{R}\). This is a set of points in \(\mathbb{R}^n\) with a constant inner product to a given vector \(\textbf{w} \in \mathbb{R}^n\). The constant \(b\) determines the offset of the hyperplane \((\textbf{w},b)\) from the origin.

Input Space \(X\) : Space of the examples \(\mathbf{x}_i\), \(i \in \lbrace 1, 2, ..., l \rbrace\). \(X\) is a subset of \(\mathbb{R}^n\). \(X\) may be a vector space of image vectors or vectors containing any data.

Output Domain \(Y\) : For binary classification of the vectors in \(X\), \(Y=\lbrace -1, 1 \rbrace\). For every \(\mathbf{x}_i \in X\), there is a label \(y_i \in Y\).

Classification function: Based on the assumption that there exists a hyperplane \((\mathbf{w},b)\) that correctly classifies all the vectors in the training data \(S\subseteq X\) , the classification function is defined as

\[\forall \mathbf{x}_i \in S, f(\mathbf{x}_i)=\mathbf{w}^T \mathbf{x}_i + b\]

For the classification \(sign(f(\mathbf{x}_i))\) is used. If \(y_i=sign(f(\mathbf{x}_i))\) this implies correct classification. The convention \(sign(0)=1\) is used.

Margin: The margin of an example \((\mathbf{x}_i, y_i)\) with respect to a hyperplane \((\mathbf{w},b)\) is given as

\[\gamma_i = y_i(\mathbf{w}^T\mathbf{x}_i +b)\]
\[\begin{split}\left\{ \begin{array}{ll} \gamma _i >0 & \mbox{ $\Rightarrow$ the classification is correct}\\ \gamma _i <0 & \mbox{incorrect classification $\Rightarrow \mathbf{w}$ has to be updated} \end{array} \right.\end{split}\]
