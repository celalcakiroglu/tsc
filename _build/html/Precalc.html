Pre-calculus¶

Number sets¶

  • Real numbers \(\mathbb{R}\): This set contains all the numbers that we deal with in calculus except complex numbers.
  • Natural numbers or positive integers \(\mathbb{N}\): This set consists of the integers \(\lbrace 1, 2, 3, ...\rbrace\)
  • Integers \(\mathbb{Z}\): \(\lbrace..., -3, -2, -1, 0, 1, 2, 3, ...\rbrace\).
  • Rational numbers \(\mathbb{Q}\): These are the numbers that we obtain by dividing one integer by another integer. In other words, for every rational number \(q\in\mathbb{Q}\), there exist two integers \(p,r\in\mathbb{Z}, r\neq 0\) such that \(q=p/r\).

Axioms of the set of real numbers \(\mathbb{R}\)¶

  • For any \(a,b\in\mathbb{R}\), \(a+b\in\mathbb{R}\) and \(a\cdot b \in \mathbb{R}\) (Closure law).
  • For any \(a,b\in\mathbb{R}\), \(a+b=b+a\) and \(a\cdot b=b\cdot a\) (Commutative law).
  • For any \(a,b,c\in\mathbb{R}\), \((a+b)+c=a+(b+c)\) and \((a\cdot b)\cdot c=a\cdot (b\cdot c)\) (Associative law).
  • For any \(a,b,c\in\mathbb{R}\), \(a\cdot(b+c)=ab+ac\) and \((a+b)\cdot c=ac+bc\) (Distributive law).

Factoring Polynomials¶

For any \(a,b,c,x\in\mathbb{R}\):

  • \(a^2-b^2=(a+b)(a-b)\)
  • \(a^2 +2ab+b^2=(a+b)^2\)
  • \(a^2 -2ab+b^2=(a-b)^2\)
  • \(a^3+b^3=(a+b)(a^2-ab+b^2)\)
  • \(a^3-b^3=(a-b)(a^2+ab+b^2)\)
  • \(x^2+(a+b)x+ab=(x+a)(x+b)\)
  • To factor \(ax^2+bxy+cy^2\), find two factors of \(ac\) that add to \(b\). Let \(ac=f_1 f_2\) such that \(f_1+f_2=b\). Then the polynomial can be written as \(ax^2+f_1xy+f_2xy+cy^2\). Since \(a=f_1f_2/c\), the polynomial can be written as
\[\begin{split}\frac{f_1f_2}{c}x^2+f_1xy+f_2xy+cy^2&=f_1f_2x^2+f_1cxy+f_2cxy+cy^2\\ &=f_1cx(f_2x+y)+cy(f_2x+y)\\ &=(f_2x+y)(f_1cx+cy)\end{split}\]

As an example let’s factor the polynomial \(3x^2+8xy+4y^2\): Since \(3\cdot 4=12=6\cdot 2 \text{ and } 8=6+2\), we have

\[\begin{split}3x^2+8xy+4y^2&=3x^2+6xy+2xy+4y^2\\ &=3x(x+2y)+2y(x+2y)\\ &=(3x+2y)(x+2y)\end{split}\]
